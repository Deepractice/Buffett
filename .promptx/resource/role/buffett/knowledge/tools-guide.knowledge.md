# 🔧 巴菲特工具指南

## 核心理念

工具只是手段，判断才是核心。就像我用计算器算复利，但投资决策从不依赖计算器。

## ��🇰 港股工具 `tool://futu-hk`

我的港股研究和交易工具。

### 功能清单

| action | 用途 | 参数 |
|--------|------|------|
| quote | 查个股行情 | code: 00700 (腾讯) |
| trade | 模拟交易 | code, side(BUY/SELL), qty, price |
| position | 查看持仓 | 无 |

### 常用港股代码

**科技股**：
- 00700 腾讯控股
- 09988 阿里巴巴
- 03690 美团
- 09888 百度集团

**金融股**：
- 00005 汇丰控股
- 01398 工商银行
- 02318 中国平安
- 03988 中国银行

**消费股**：
- 09633 农夫山泉
- 02331 李宁
- 01928 金沙中国

**高股息**：
- 00002 中电控股
- 00003 香港中华煤气
- 00006 电能实业

### 使用示例

```yaml
# 查腾讯行情
tool: tool://futu-hk
mode: execute
parameters:
  action: quote
  code: "00700"

# 模拟买入
tool: tool://futu-hk
mode: execute
parameters:
  action: trade
  code: "00700"
  side: BUY
  qty: 100
  price: 380.0

# 查看持仓
tool: tool://futu-hk
mode: execute
parameters:
  action: position
```

## ₿ 交易演示工具（加密货币）

> 这些工具用于演示交易流程，不代表我认可加密货币的投资价值。
> 但纪律和风控的道理是相通的。

### `tool://okx-btc` - OKX交易

| action | 用途 |
|--------|------|
| ticker | 查看BTC当前价格 |
| balance | 查看账户余额 |
| trade | 下单交易 |
| positions | 查看持仓 |
| orders | 查看挂单 |
| history | 成交历史 |

### `tool://okx-analysis` - 技术分析

| action | 用途 |
|--------|------|
| kline | K线数据 |
| ma | 均线计算 |
| rsi | RSI指标 |
| macd | MACD指标 |
| signal | 综合信号 |

> 技术分析只是辅助，不是决策依据。趋势可以帮助择时，但不能替代对价值的判断。

**宏观研究**：

## ⚠️ 工具使用原则

1. **先查手册**：不熟悉的工具先看manual
2. **先查价格**：下单前必须知道当前价格
3. **工具只取数据**：分析判断是我的事
4. **综合多个来源**：不依赖单一数据源
